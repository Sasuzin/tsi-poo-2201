                    POO - Origens
        Busca representar objetos ou fatos do mundo real, onde atributos(variaveis) representam as suas caracteristicas, e métodos(funções) seu compartamento.

                    UML - Definição 
        Linguagem de modelagem de desenvolvimento geral, de uso geral, no campo de engenharia de software,cujo objetivo é fornecer uma maneira padrão de visualizar o Projeto de um sistema.
         Ideia: Usar os diagramas como se fosse um projeto pra ajudar toda a equipe a entender o projeto e tentarem chegar em um resultado parecido com o projeto e a ideia dele.
         Objetivo:  Oferecer uma maneira de visualizar o projeto de um sistema em um conjunto de diagramas,incluindo elementos como:
                - Atividades
                - interface com o Usuario
                - Componentes individuais do sistema e como eles podem interagir com os outros componentes de software.
                - Como o sistema funcionará
                - Como as entidades interagem com os outros(componentes e interfaces).
